# Security Policy

## Supported Versions

Only the latest release receives security patches.

## Reporting a Vulnerability

Use [GitHub's private vulnerability reporting](https://github.com/glauberlima/claude-code-statusline/security/advisories/new) to report security issues. Do **not** open a public issue.

**Expected response:** acknowledgement within 48 hours, fix or mitigation within 90 days depending on severity.
